# Assignment3_NGO_RacialInjustice
User Instruction (given)

First come with the Login Page. There are provided a select login option either User or Admin:
If login as User, select loginOption as "User", the username and password must exist in the textFile first, only the User can log in using by their personal username and password.
For the situation that the username and password that not exist in txtFile means that the user is First Time User, they need to click Sign Up Button to
register their username and password for log in as User. To log in successful, they must tick to agree the Terms & Conditions and Policy Private.
Click Submit to upload the register information to store in the txtFile. Then back to the Login Page, only can log in to view the communityBackground (SumWithUs).
If user need to see the Event, can at the SumWithUs windows click "Click Here To View More Recent Event" to proceed next windows (Event).
At the Event frame, user can click the radioButton eventType to view the Event Poster and each event Details.
To do registration, user need to click which event they need to participate to proceed the Registration windows (Participant).
Fill all the information to proceed registration Event and also the donation.
Click Donation and select the Confirm Button to proceed a Message that display "Donation Done Successfully!"
then only will jump to Advertisement window. Then click exit button will jump back to login page then click exit again only exit the program.
